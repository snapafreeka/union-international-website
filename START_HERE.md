## ⚡ Super Quick Start (5 Steps)

If you just want to get started NOW:

```bash
# 1. Install pnpm (if not installed)
npm install -g pnpm

# 2. Navigate to project folder
cd /path/to/union-international-website

# 3. Install dependencies
pnpm install

# 4. Start the server
pnpm dev

# 5. Open browser
# Visit: http://localhost:5173
```

**That's it!** Your website should be running.
