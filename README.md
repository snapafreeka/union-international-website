## 🌟 About

Professional website for **Union International Limited** - Licensed Customs Agent #1 in Kenya, providing:

- ✅ Imports & Exports Services
- 📦 Electronic Sealing Solutions  
- 🚚 Electronic Cargo Tracking & Fleet Management
- ⚙️ Managed Services & System Integration
- 📋 Compliance & Bids Consultancy
- 🔐 Type Approval Services

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

Visit: **http://localhost:5173**

## 🏗️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS v4
- **Routing**: React Router v6
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Analytics**: Google Analytics 4

## 🔧 Configuration

### Environment Variables
Create `.env` file:

```env

VITE_GTM_CONTAINER_ID=GTM-XXXXXXX
VITE_CLARITY_PROJECT_ID=XXXXXXXXXX
```

### Google Analytics
Already configured in `src/app/components/Analytics.tsx`
